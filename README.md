# Bored API

![Demo Gif](https://i.pinimg.com/originals/7d/bd/e6/7dbde6ca4b7d29916fe68af9127ee1ba.gif)

A simple API built with Spring Boot that generates a new task every time it is called.

## Prerequisites

- Java 8 or higher
- Maven
